##  [Leaves](http://claudetech.github.io/leaves/)

### Static website build tool

Depends only on NodeJS.

Uses:

* Yeoman
* Grunt
* Bower
* Jade (or EJS)
* Stylus (or less or plain CSS)
* Coffee (or plain JS)

Install with:

```sh
$ npm install -g leaves
$ leaves setup
```

Checkout [the docs](http://claudetech.github.io/leaves/) for more info.
